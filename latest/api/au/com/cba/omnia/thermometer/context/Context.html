Context

Related Doc: package context

case class Context(config: Configuration) extends Product with Serializable

A Context is a HDFS wrapper that sacrifices all safety, composability and sanity in order to give the illusion that you may be able to test a scalding job. Methods on thermometer return raw values that are appropriate to use with specs2 matchers.
